# Student Attendance Management System (Java)

## Detailed Description

The **Student Attendance Management System** is a Java console program designed to track student attendance.

Users can register students, mark them present, and view attendance records. The system stores student information and the number of days they have attended.

This project demonstrates key **Object-Oriented Programming (OOP)** concepts and data management using Java collections.


## Features

*  Register new students
*  View all registered students
*  Mark student attendance
*  Track number of days present
*  View attendance record for a specific student


## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList Collection
* Console Input / Output

## How to Run

Install Java (JDK 8+)

Compile the program

javac Main.java

Run the program

java Main
